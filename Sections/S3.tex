Um die Email-Adressen der Nutzer zu speichern wird eine XML-Datei auf einem Simple Storage Service(S3) angelegt. Dies ist ein von AWS entwickelter Dienst zur einfachen Ablage von Dateien. In der Anwendung wird ausschließlich auf dem privaten Bucket \verb+flightradaremail+ gearbeitet. Der zugriff erfolgt nur über das AWS-SDK vom Server und über einen Lambda-Service.
