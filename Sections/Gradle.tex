Build-Tools machen das automatische Bauen von Anwendungen einfach. Bei der Auswahl des Build-Tools fiel die Wahl auf Gradle. Dependency-Management und das Erstellen eigener Tasks, die während des Builds ausgeführt werden ist möglich. So wurde in dieser Anwendung ein Task \verb+integrationTests+ erstellt. Dieser dient zum Ausführen der Integration-Tests. Hierbei wird das Schreiben und Lesen auf der Datenbank getestet.
