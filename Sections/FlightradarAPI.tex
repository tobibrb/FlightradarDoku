Bei Flightradar24 handelt es sich um einen Live-Tracking-Service für Flugzeuge. Hierzu werden Daten von vielen verschiedenen Diensten zusammen gefasst. Die Daten werden in einer nicht öffentlichen API zur Verfügung gestellt. Diese wird von den Produkten und Anwendungen der Flightradar24 AB verwendet um die aktuellen Flugdaten zur Verfügung zu stellen.
